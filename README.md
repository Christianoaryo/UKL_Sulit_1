# Program ini digunakan untuk menghitung rata-rata nilai siswa berdasarkan jumlah siswa dan nilai yang dimasukkan oleh pengguna
1.Scanner untuk input jumlah siswa
2.Jika input bukan bilangan bulat atau kurang dari 1, program akan menampilkan pesan kesalahan dan meminta input ulang.
3.menggunakan perulangan for untuk menambahkan jumlah siswa setelah menginputkan nilai
4.Di dalam perulangan, terdapat while (true) yang digunakan untuk validasi inputan
  -Jika input bukan angka (!input.hasNextDouble()), program akan menampilkan pesan kesalahan dan meminta input ulang.
5.Jika angka yang dimasukkan kurang dari 0 atau lebih dari 100, program juga akan menolak dan meminta input ulang.
6.- Setelah input valid diterima, nilai tersebut ditambahkan ke variabel totalNilai untuk digunakan dalam perhitungan rata-rata.
